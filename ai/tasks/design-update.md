# Task: 디자인·UI 변경

## 사용 방법
```
Agent: ui-design-lead
Skill: design-system-ui (또는 style-polish)

다음 화면/컴포넌트의 디자인을 바꿔줘:

[대상]
- 경로 또는 컴포넌트명: 예) src/components/Landing/ ...

[원하는 방향]
- 레이아웃 / 색 / 타이포 / 여백 / 버튼 스타일 등

[참고]
- 스크린샷, 피그마 링크, 또는 “OO 사이트 상단 느낌” 등
```

## 구현 위주로 바로 손댈 때
```
Agent: nextjs-builder
Skill: design-system-ui

[파일] …
[변경 내용] …
```

## 체크리스트
- [ ] `[lang]` 라우팅·API·서비스 레이어는 변경하지 않았는지
- [ ] 새 의존성 추가 없음
- [ ] 모바일에서 확인할 포인트 명시 여부

# Skill: 코드 리뷰

## 목표
ZEBA 프로젝트 기준으로 코드 품질 검토

## 검토 항목

### 1. Next.js / React
- [ ] Server/Client 컴포넌트 구분 적절한가
- [ ] `[lang]` 라우팅 구조 준수하는가
- [ ] 불필요한 `"use client"` 없는가
- [ ] React 19 호환성 문제 없는가

### 2. TypeScript
- [ ] `any` 타입 사용하지 않았는가
- [ ] Props 인터페이스 정의 있는가
- [ ] `Promise<Params>` 패턴 사용하는가 (Next.js 16)

### 3. 성능
- [ ] 불필요한 리렌더링 없는가
- [ ] Framer Motion 애니메이션 최적화되었는가
- [ ] 이미지 최적화 (`next/image`) 사용하는가

### 4. 코드 품질
- [ ] 중복 코드 없는가
- [ ] 컴포넌트 크기 적절한가 (200줄 이하 권장)
- [ ] 재사용 가능한 부분 추출했는가

### 5. 스타일
- [ ] Tailwind v4 문법 사용하는가
- [ ] 인라인 스타일 최소화
- [ ] 기존 디자인 시스템 활용하는가

## 출력
1. 문제점 목록 (우선순위별)
2. 개선 제안
3. 수정 코드 (필요시)
